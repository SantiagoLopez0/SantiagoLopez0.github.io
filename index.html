My Portfolio

I do this portfolio as a personal project based on my knowledge and skills in Web Development. In the same way, I made it thinking the best and easiest form to show my professional career path in a different way than my Resume.

This page was made with HTML, CSS, and Vanilla JavaScript. I coded it like that because I want to focus on the semantics and write the cleanest and clearest form possible. Data Recorders

This page was build in two versions. It has as a goal to record data client easily and intuitively. I participated in this project as Frontend Developer implement technologies like JQuery, Bootstrap, and Razor coded in tandem with a backend made in C#.

In the first image, you can see the earliest version. It was made based on wizard design since it has to show every step one by one to complete the main form, validating while the client is filling each field.

In the next image, you can see the latest version. That was modified to record the same data that the first version but in only one step, validating all fields at the same time.

Medical Appointment Scheduling Template

This template is made to announce the services of a medical consultant for subsequently can schedule a medical date with a specialist on the same page. The design of the site facilitates taking the appointment remotely.

I implement technologies like JQuery, Bootstrap, and razor. Besides, I worked integrating API's to the site.
