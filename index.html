Este proyecto personal fue creado con el objetivo de reforzar mis conocimientos en las bases del desarrollo web. Al mismo tiempo lo hice pensando en poder crear una forma más sencilla de ver los proyectos en los cuales he trabajado y plasmar mi trayectoria en el ámbito profesional. En primer lugar, lo hice de esta manera con el objetivo de enfocarme en la semántica y en que el código que escribiera fuera lo más limpio y claro posible. Sin embargo, más adelante pienso implementar tecnologías como React.JS o algún otro framework para aumentar la interactividad en este sitio.

Registradores de Datos

Esta pagina web esta hecha en 2 versiones. Tienen como objetivo recolectar datos de clientes de forma fácil e intuitiva. En este proyecto participe como desarrollador frontend, implementando tecnologías como jQuery, Bootstrap y Razor, uniéndolas al backend hecho con C# (.NetCore).

La primera imagen que puedes ver evidencia la primera versión hecha de este sistema de registro de datos. Se hizo en forma de Wizard ya que se necesitaba que se pudieran evidenciar la cantidad de pasos necesarios para completar todo el formulario, validando paso por paso los campos requeridos para poder continuar.

En la segunda imagen puedes ver la segunda versión, modificada desde la primera con el objetivo de poder registrar los mismos datos que se necesitaban en la primera versión, pero en un solo paso. De igual forma se validan los campos requeridos para poder enviar el formulario.

Plantilla para agendamiento de citas medicas

Esta es una plantilla hecha específicamente para dar a conocer los productos y servicios que ofrece una consultora médica, para posteriormente poder agendar una cita medica desde el mismo sitio con un doctor, además de poder tomar dicha cita de forma virtual desde esta misma página.

Ya que esta pagina ofrece la posibilidad de ver datos específicos sobre cada paciente, se implemento un sistema de inicio de sesión y registro de cuenta, así mismo de una cantidad especifica de formularios donde se puede preguntar todo lo necesario para poder agendar la cita.

En esta plantilla implemente tecnologías como JQuery, Bootstrap y Razor. Además, trabaje con integración de API’s al sitio web como Full Calendar. El backend de este sitio web esta hecho con C# (.Net Core Framework) y aún se sigue mejorando de forma independiente ya que no está listo para lanzar a producción.
